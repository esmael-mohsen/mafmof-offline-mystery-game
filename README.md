## Overview

**Mafmof** is an offline-first Android game built for group mystery nights.  
Players receive secret roles, follow staged evidence, discuss accusations, vote, and reach a final reveal.

The app is designed around:

- Arabic-first RTL experience.
- Cinematic dark/crimson visual identity.
- Offline play with local case data.
- 5 to 8 player variants.
- Private role reveal flow.
- Host dashboard, staged clues, voting, and final reveal.

## Installation For Testers

1. Open the GitHub Release link on your Android phone.
2. Download `mafmof-v1.0.0-build1.apk`.
3. If Android asks for permission, allow **Install unknown apps** for the browser or file manager.
4. Tap the APK and install.
5. Open **Mafmof** from the app launcher.

> This APK is privately distributed. Only users with access to this private repository can download it from GitHub.
